# **App Name**: Dark Pattern Detective

## Core Features:

- Text Analyzer: Analyze website text for dark patterns such as fake urgency, hidden costs, forced choices, and misleading language.
- Dark Pattern Detection: Use AI to automatically detect potential dark patterns in the text.
- Pattern Highlighting: Highlight the detected dark patterns in the user interface with explanations.
- Confidence Scoring: Provide a confidence score for each detected pattern to indicate the likelihood of it being a dark pattern.
- Text Input: Provide a text input area to enter the desired text.
- Analysis Output: Output the analysis results

## Style Guidelines:

- Primary color: Soft blue (#A0CFEC) to inspire trust and reliability, drawing from the concept of authority.
- Background color: Light gray (#F0F4F7), a desaturated version of the primary, provides a neutral backdrop.
- Accent color: Warm orange (#E78A4E), analogous to blue, emphasizes warnings and detected patterns with high contrast.
- Font: 'Inter', a sans-serif font known for its clarity and readability, making it ideal for analytical text.
- Use clear, recognizable icons to represent different types of dark patterns.
- Clean, well-organized layout with clear separation between input and analysis sections.
- Subtle animations to highlight detected patterns and guide the user through the analysis.
